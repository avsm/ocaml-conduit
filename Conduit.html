Module Conduit

module Conduit: sig .. end
Interface for establishing reliable stream-oriented connections.

This library abstracts the concerns of establishing connections to peers that may be running within the same host (e.g. in another virtual machine) or on a remote host via TCP. It consists of two main components that are responsible for:

  • Establishing a single connection to a remote peer that is identified by an Conduit.endp address.
  • Resolving URIs into a list of Conduit.endp addresses that can then be connected to by the transport layer.
To ensure portability, the Conduit.endp addresses are translated into concrete connections by separate modules that target Lwt_unix, Async and Mirage.
